/h2h/profile

Endpoint untuk mengambil informasi profil pengguna berdasarkan API Key.

Endpoint 

GET https://www.v-pedia.web.id/h2h/profile?apikey=V-Pedia-xxxxx

Authentication

Endpoint ini membutuhkan API Key yang valid dalam header request.

Authorization: Bearer YOUR-APIKEY

Parameters

Parameter Type Required Description
apikey String Yes API Key milik Anda

Success Response

Berikut adalah contoh response yang akan diterima ketika request berhasil:

Error Responses

Internal Server Error

Terjadi kesalahan pada server

{
  "success": false,
  "message": "Internal server error"
}
Missing API Key

Parameter apikey tidak diisi

{
  "success": false,
  "message": "Parameter \"apikey\" harus di isi"
}
Account Not Verified

Akun H2H belum diverifikasi

{
  "success": false,
  "message": "Akun belum terverifikasi. Silakan verifikasi akun terlebih dahulu."
}

Code Examples

Berikut adalah contoh implementasi endpoint ini dalam berbagai bahasa pemrograman:

JavaScript (Fetch API)

fetch("https://www.v-pedia.web.id/h2h/profile", {
  method: "GET",
  headers: {
    "Authorization": "Bearer V-Pedia-xxxxxxx"
  }
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error("Error:", error));

Python (Requests)

import requests

headers = {
    "Authorization": "Bearer V-Pedia-xxxxxxx"
}

response = requests.get(
    "https://www.v-pedia.web.id/h2h/profile",
    headers=headers
)

print(response.json())

cURL

curl -X GET "https://www.v-pedia.web.id/h2h/profile" \
  -H "Authorization: Bearer V-Pedia-xxxxxxx"

    "Authorization": "Bearer V-Pedia-xxxxx"

)

# Cek apakah response sukses

    data = response.json()
    print("Data kategori:", data)
else:
    print("Gagal mengambil data. Status:", response.status_code)
    print("Respon:", response.text)

Authorization: Bearer YOUR-APIKEY

Parameters

Parameter Type Required Description
code String Yes Kode produk (contoh: ML3 untuk 3 Diamond Mobile Legends)
target String Yes ID target (contoh: ID pengguna & zone ID untuk game, format: "USERID|ZONEID")
apikey String Yes API Key milik Anda

  "message": "Parameter \"code\" harus di isi"
}
Missing Target

Parameter target tidak diisi

{
  "success": false,
  "message": "Parameter \"target\" harus di isi"
}
